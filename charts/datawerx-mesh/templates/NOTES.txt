Next steps:

1. Peer with another cluster by creating a MeshPeer (one per remote cluster):

     apiVersion: networking.datawerx.io/v1alpha1
     kind: MeshPeer
     metadata: { name: cluster-b }
     spec:
       clusterID: cluster-b
       publicKey: "<remote wg public key>"
       endpoint: "<remote host:51820>"
       podCIDRs: ["10.245.0.0/16"]
       serviceCIDRs: ["10.97.0.0/16"]

2. Export a Service for cross-cluster discovery:

     kubectl apply -f - <<EOF
     apiVersion: multicluster.x-k8s.io/v1alpha1
     kind: ServiceExport
     metadata: { name: <svc>, namespace: <ns> }
     EOF
